This module allows realtime colaboration on Drupal nodes.

REQUIREMENTS
------------

The module uses the Google Realtime APIs client library for PHP. You can find more 
information on it at the following url: https://developers.google.com/drive/realtime

The module also has dependencies with the following contributed ones:
- Libraries

INSTALLATION
------------

1. Download the module from drupal.org, and place it somewhere in the
'sites/all/modules/' directory.
